TomLeafVision is a deep learning-based web application designed to detect tomato leaf diseases such as Early Blight and Late Blight. The system uses a custom Convolutional Neural Network (CNN) model trained on plant leaf images to classify diseases accurately.

The application is built using Flask for the backend and provides a user-friendly interface where users can upload leaf images and receive instant predictions.

Key Features:
- Real-time disease detection
- Custom CNN model
- Image upload and prediction system
- User-friendly web interface
- Accurate classification of Healthy, Early Blight, and Late Blight leaves

Technologies Used:
- Python
- TensorFlow / Keras
- Flask
- HTML, CSS

This project helps farmers and researchers in early detection of plant diseases, improving crop health and productivity.
